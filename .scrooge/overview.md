**AA Grapher** is a static HTML+JS scatter-plot dashboard for visually exploring hundreds of AI models by their benchmark scores, pricing, and latency.

The pipeline has two phases. First, `clean.py` scrapes a raw Next.js RSC payload (the `providers` file) — not standard JSON — using regex-anchored `json.JSONDecoder` calls to extract ~900 model×host entries. It flattens each model's nested evaluation fields into dot-separated keys, aggregates multiple host providers per model by taking the **median** of price/speed metrics, computes a derived `intelligence_index_run_cost`, and writes `data.js` (two exported constants: `FIELDS` — an array of objects each with `name`, `pretty_name`, `direction`, `scale` — and `DATA`). Second, `index.html` is a self-contained Plotly.js application that reads `data.js` and renders an interactive scatter plot with log/linear axes, 2D and 3D Pareto frontier filtering (sign-adjusted so "larger is better" on every axis), and color-mapped metrics with automatic log scaling when the dynamic range exceeds 100×. Field direction defaults are heuristic: any field name containing "price", "cost", "time", "latency", "hallucination", or "tokens" is treated as `min` (lower is better); `context_window_tokens` and `non_hallucination` are `max`. The project owns no server, no build step — open `index.html` in a browser and it works.
